# Skill Network Analysis

This project models professional skills as a dependency network using co-occurrence data from 20,000+ skills and millions of pairs.  
- **Tech stack:** Python, Pandas, NetworkX, Matplotlib  
- **Methods:** Jaccard similarity, degree & betweenness centrality  
- **Findings:** Identifies gateway skills (business, software, automation) and monopoly skills (healthcare, control theory, informatics).  
- **Impact:** Provides insights for career development, education programs, and labor market analysis.  

The repository includes cleaned datasets, analysis scripts, and visualizations highlighting the structure of skill interdependencies.
